# 📱 MediScan AI 使用说明

## 🎯 应用简介

MediScan AI 是一个智能医疗报告解读助手，可以帮助你：
- 理解复杂的医疗检查报告
- 识别医疗影像中的关键信息
- 获取通俗易懂的健康建议

⚠️ **重要提示**：本工具仅供参考，不能替代专业医疗诊断！

## 🚀 快速开始

### 首次使用

1. **获取 API Key**（免费）
   - 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
   - 使用 Google 账号登录
   - 点击 "Create API Key" 创建密钥
   - 复制生成的密钥（格式：`AIzaSy...`）

2. **在应用中配置**
   - 打开 MediScan AI 应用
   - 首次使用会提示输入 API Key
   - 粘贴你的密钥并保存
   - 密钥会安全地保存在浏览器本地

### 开始使用

1. **上传文件**
   - 点击 "上传文件" 按钮
   - 选择医疗影像或报告截图
   - 支持格式：JPG, PNG, PDF 等

2. **查看分析结果**
   - AI 会自动分析图像内容
   - 提供详细的解读和建议
   - 标注异常指标（如有）

3. **对话追问**
   - 可以继续提问获取更多信息
   - 例如："这个指标偏高会有什么影响？"
   - AI 会基于之前的分析回答

4. **语音播报**
   - 点击播放按钮听取解读
   - 适合老年人使用
   - 可以暂停/继续播放

## 💡 使用技巧

### 拍摄建议
- 📸 **光线充足**：在明亮环境下拍摄
- 📄 **画面完整**：确保文字清晰可见
- 🎯 **焦点准确**：避免模糊或倾斜
- 📐 **避免反光**：不要有强光照射

### 提问技巧
- ✅ 具体明确："甘油三酯 2.5 算高吗？"
- ✅ 询问建议："如何降低这个指标？"
- ✅ 追问细节："为什么会出现这种情况？"
- ❌ 避免笼统："我的报告怎么样？"（应该先上传图片）

## 🎨 功能介绍

### 1. 医疗影像分析
支持的影像类型：
- X 光片（胸片、骨骼等）
- CT 扫描
- MRI 核磁共振
- B 超报告
- 心电图

### 2. 检验报告解读
支持的报告类型：
- 血常规
- 生化检查
- 尿常规
- 肝功能、肾功能
- 血糖、血脂
- 甲状腺功能
- 肿瘤标志物

### 3. 健康建议
AI 会提供：
- 🔍 异常指标解释
- 💊 可能的原因分析
- 🏃 生活方式建议
- 👨‍⚕️ 是否需要就医

## 🔒 隐私与安全

### 数据处理
- ✅ 图片仅用于分析，不会保存到服务器
- ✅ 对话记录存储在本地浏览器
- ✅ 可随时清除历史记录
- ✅ API Key 加密保存在本地

### 清除数据
如需清除所有数据：
1. 打开浏览器设置
2. 清除浏览数据
3. 选择 "Cookie 和其他网站数据"
4. 重新打开应用即可

## ❓ 常见问题

### Q: 免费吗？
A: 应用本身免费。Google AI API 有免费额度，足够个人使用。

### Q: 准确吗？
A: AI 分析仅供参考，不能替代医生诊断。如有疑虑，请咨询专业医生。

### Q: 支持哪些语言？
A: 目前主要支持中文和英文报告。

### Q: 可以离线使用吗？
A: 不可以。需要联网调用 AI API 进行分析。

### Q: API Key 会过期吗？
A: Google AI 的免费 Key 不会过期，但有使用额度限制。

### Q: 分析速度慢？
A: 可能原因：
- 图片过大（建议压缩到 2MB 以内）
- 网络不稳定
- API 配额已用完（等待重置）

## 🛠️ 技术支持

遇到问题？
1. 查看浏览器控制台错误信息（F12 打开）
2. 确认 API Key 是否正确
3. 检查网络连接
4. 尝试刷新页面或清除缓存
